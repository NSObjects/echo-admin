# Triage Labels

工程技能使用五个 canonical triage role 描述 issue 状态。本文件把这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| Label in mattpocock/skills | Label in our tracker | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估这个 issue |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已经明确，agent 可以直接接手 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色，例如“apply the AFK-ready triage label”，使用表格右侧对应的标签字符串。

如果仓库以后调整标签命名，只修改右侧 `Label in our tracker` 列，不要修改左侧 canonical role。
